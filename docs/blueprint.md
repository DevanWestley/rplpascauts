# **App Name**: Advocacy Alley

## Core Features:

- Admin/Non-Admin Login: Secure login for admins and regular users with role-based access control.
- Create Petition: Authenticated users can create petitions with title, description, target signatures, category, attachments, and expiry date.
- Visibility Options: Set petition visibility to public or private.
- Sign Petition: Users can sign petitions with name, email, optional location, and comments. Signature count updates in real-time.
- Petition Page: Displays petition details, signature count, progress bar, supporter list, and a sign button.
- Basic Authentication: Social login (Google/Facebook) or email+password for creators/admins; option for signees to sign without an account using email verification.
- Share Petition: Generate social media share buttons, short links, copy link, and embed codes for wider distribution.

## Style Guidelines:

- Primary color: Dark gray (#333333) for a serious and professional feel.
- Background color: Light gray (#F0F0F0) for a clean, neutral backdrop.
- Accent color: Medium gray (#808080) to provide subtle contrast without disrupting the monochromatic theme.
- Body and headline font: 'Inter' (sans-serif) for a modern and readable interface.
- Use minimalist, line-style icons for key actions and categories.
- Emphasize a clean, content-focused layout with ample whitespace.
- Subtle transitions and progress animations to provide feedback without being distracting.
